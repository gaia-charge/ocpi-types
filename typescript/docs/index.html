Module ocpi-types-typescript[src]

ocpi-types-typescript

Auto-generated TypeScript definitions for the OCPI protocol.

Installation

npm install --save-dev ocpi-types-typescript

Usage

You can import all defined types as so:

import type { Locationinterface LocationLocation } from 'ocpi-types-typescript/v2.1.1';

const myLocation: Locationinterface LocationLocation;
